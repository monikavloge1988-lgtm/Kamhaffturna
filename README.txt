# Tournament App Node.js Backend & SQL

We have created the Node.js backend using Express and MySQL according to your hosting environment instructions, and we compiled the `database.sql` file.

To run this on your hosting:
1. Import `database.sql` into your cPanel phpMyAdmin database (`yaarwinn_kff2`).
2. Put this code on your host, ensuring your Node.js app runs `server.js`.
3. Create a folder named `public` in this directory.
4. Paste your user application HTML into `public/index.html`.
5. Paste your admin application HTML into `public/admin.html`.

You can now download all these files directly!

### To Download as a ZIP:
1. Click the **Settings/Menu** gear icon in the top right of this AI Studio window.
2. Click **Export to ZIP** (or Export to GitHub).
3. The zip file will contain `server.js`, `package.json`, and `database.sql`.
